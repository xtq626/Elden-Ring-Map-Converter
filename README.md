## 工具简介
**ER→NR 地图转换工具**是一款Windows程序，用于将《艾尔登法环》地图数据移植至《艾尔登法环：黑夜君临》。工具提供全流程转换功能：涵盖MSB格式转换、怪物参数重映射、碰撞几何体对齐、事件脚本合并等。模组制作者无需手动修改十六进制文件，就能把法环场景（怪物、建筑、碰撞体、剧情事件）完整迁移到黑夜君临的探索地图内。

## 安装步骤
1. 双击`法环转黑环工具.exe`启动程序，软件为独立封装包，无需额外安装.NET运行库。
2. 将游戏本体提取的`oo2core_9_win64.dll`放到程序同目录，该文件是高精度贴图匹配功能的必需依赖。
3. 使用**ModEngine3**加载修改后的地图文件进入游戏。

## 核心功能
### 格式转换
法环地图采用MSBE格式，黑夜君临使用MSB_NR，二者底层结构完全不兼容。该功能页可将法环`.msb.dcx`转换为黑夜君临可用地图，完整保留7大类数据：地形、怪物、场景物件、碰撞体、玩家出生点、虚拟道具、虚拟怪物，并自动适配全部参数。
内置别名对照表自动处理字段命名差异（如`NPCParamID`统一修正为`NpcParamId`、`WalkRouteName`改为`PatrolRouteName`）。同时自动修复两类会导致游戏崩溃的bug：碰撞体MapID为`[0,0,0,0]`（黑夜君临要求改为`[-1,-1,-1,-1]`）；9开头五位数碰撞块（引用黑夜君临不存在的外部碰撞合集）。
**使用方法**：选择法环`.msb.dcx`文件，设置输出路径，点击转换。

### 合并地图
适用于将整张法环区域拼接进已有黑夜君临探索地图。该功能页会把源地图所有物件、模型追加到目标地图，自动重命名冲突条目并同步更新全部内部资源引用。
可选勾选功能可将导入怪物绑定至目标地图主流怪物分组，黑夜君临的副本生成系统才能正常刷新怪物。
**使用方法**：选择黑夜君临地图作为基底、待导入的法环地图为源文件；可填入「地图碰撞合并」功能生成的碰撞映射表，最后点击合并。

### 复制模型资源
格式转换仅修改地图配置文件，模型与贴图素材仍需从本地法环客户端提取。该页面读取MSB，检索所有引用的AEG模型，在游戏目录批量匹配对应文件：几何体`.geombnd`、高低精度碰撞`_h/_l.geomhkxbnd`、AET贴图包，并按规范目录结构输出到模组文件夹。
三种贴图复制模式：精准模式（仅复制模型对应编号贴图包）、深度精准模式（读取FLVER材质解析共用贴图，依赖oo2core文件）、全文件夹模式（复制完整贴图目录，冗余度高但兼容性最好）。
**使用方法**：加载地图文件，指定法环游戏根目录为素材源，设置模组输出路径，选择贴图模式后执行复制。

### 合并EMEVD地图事件
两地的剧情事件分属独立`.emevd.dcx`，无法直接替换。该功能读取两份事件文件，将法环事件追加至黑夜君临基底，自动跳过重复事件ID，并生成日志记录合并与跳过条目。
**使用方法**：基底选择黑夜君临事件文件、源文件选择法环事件文件，设置输出路径后合并。

### 合并公共函数common_func
专门处理`common_func.emevd.dcx`——两地共用的全局脚本库。法环专属逻辑（赐福激活、NPC对话触发等）必须导入黑夜君临公共函数库，事件才能正常调用。工具仅追加缺失函数，保留黑夜君临原有内容。
**使用方法**：基底选黑夜君临公共函数文件，源文件选法环版本，执行合并。

### 合并地图碰撞
碰撞几何体独立存放于`.hkxbhd`+`.hkxbdt`配对文件，需单独合并。该页面整合两地高低精度碰撞包，重命名冲突碰撞体并输出碰撞映射表，映射表可导入「合并地图」页面，同步更新地图内碰撞引用。
**使用方法**：分别加载法环、黑夜君临的碰撞配对文件，设置输出路径完成合并，保存生成的映射表备用。

### 自动对齐碰撞
若导入建筑存在大幅度坐标偏移（例如Y轴抬高1239单位），碰撞模型需要同步平移。黑夜君临会将碰撞坐标固化至世界空间，地图碰撞体叠加偏移会引发双重位移崩溃，因此平移后必须重置地图内碰撞坐标。
功能全自动完成：读取两份地图计算偏移量、平移HKX碰撞顶点、合并至黑夜君临碰撞包、输出修正地图（采用黑夜君临标准XZY旋转轴序）。
**使用方法**：加载法环碰撞包、黑夜君临碰撞包、合并后地图、原始法环地图，点击对齐合并，用生成的碰撞文件与地图替换模组内原文件。

### 修复旋转碰撞
三轴均带旋转的碰撞体（如`h014000_1`）若采用XYZ旋转顺序，位置会大幅偏移；纯Y轴旋转误差较小，复合旋转最高偏移百单位。该页面仅修正异常碰撞体旋转，转换为官方标准XZY欧拉顺序。
**使用方法**：加载合并完成的地图、原始法环地图（旋转数据源），设置输出路径执行修复。

### 修复NPC参数
从法环导入的怪物参数ID仅适配原版，黑夜君临参数库无对应条目，会导致怪物无血量、无AI、不掉落道具。工具按模型编号前缀匹配黑夜君临相近怪物参数（例：`c1234`模型匹配`1234`开头参数），优先选用带道具掉落的参数条目。
二次优化模式可将无掉落的匹配条目替换为同模型带掉落配置。法环独有BOSS、精英怪无匹配参数时，会写入日志供手动调整。
**前置要求**：使用Smithbox参数编辑器，从黑夜君临`regulation.bin`导出`NpcParam.csv`与`NpcThinkParam.csv`；表格首列为参数ID，首行为表头。
**使用方法**：加载合并地图、两份参数表格，设置输出路径，可选开启二次优化后修复。

### 修复碰撞/怪物MapID
旧版本转换地图专用独立修复工具。碰撞体MapID为`[0,0,0,0]`会触发崩溃；怪物MapID不为`-1`会导致怪物不刷新。该功能就地修复文件，无需完整重走转换流程。

### 批量顶点结构转换 / 地图块转换
法环FLVER模型顶点布局与黑夜君临渲染器不兼容。顶点转换功能批量修改目录内所有模型包的顶点声明；地图块转换额外给地形模型写入目标地图ID。两项操作全部完成后，导入模型才能正常渲染。

## 使用必备条件
1. 拥有正版《艾尔登法环》与《艾尔登法环：黑夜君临》
2. ModEngine3，用于加载修改后的游戏文件
3. Smithbox：查看地图文件、导出NPC参数表格
4. 程序同目录放置`oo2core_9_win64.dll`（从游戏本体复制），深度精准贴图模式必需
